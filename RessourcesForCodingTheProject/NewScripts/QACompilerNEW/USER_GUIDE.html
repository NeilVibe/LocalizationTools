1. Installation

Download & Install

Download QACompiler_Setup.exe from GitHub Releases and run it.

No Admin Rights Required!
The installer is designed for high-security environments. Installs to your Desktop by default (NOT Program Files).

Installation Steps

1 Run the installer - Double-click QACompiler_Setup.exe

2 Select your Perforce Drive

Drive Selection Screen

The installer will ask which drive contains your Perforce workspace:

F: (Default) - Most common
D: or E: - If your Perforce is on a different drive

Enter just the letter (e.g., F, D, E)

3 Choose Install Location

Recommended: Keep the default location (Desktop)

The program will be installed to: C:\Users\[YourName]\Desktop\QACompiler\

This is a portable installation - everything stays in one folder.

4 Complete Installation - Click Install and wait for completion

5 Launch - Check "Launch QA Compiler Suite" and click Finish

6. Progress Tracker

The _TRACKER.xlsx file tracks all tester progress and manager stats.

Sheets

Sheet Purpose
DAILY Day-by-day progress per tester
TOTAL Overall statistics and workload analysis
_DAILY_DATA Raw data (hidden)

TOTAL Tab Structure

The TOTAL tab has three color-coded sections:

Blue: Tester Stats Green: Manager Stats Orange: Workload Analysis
Done, Issues, No Issue, Blocked, Korean Fixed, Reported, NonIssue, Checking, Pending Actual Done, Daily Avg, Type, Days Worked, Assessment

Workload Analysis Columns

Column Source Description
Actual Done Auto-calculated Done - Blocked - Korean
Daily Avg Auto-calculated Actual Done / Days Worked
Type TesterType.txt "Text" or "Gameplay"
Days Worked Manual entry Manager fills in
Tester Assessment Manual entry Manager's quality notes
Days Worked and Tester Assessment are preserved!
When you rebuild Master files, these manual entries are automatically preserved.

7. Troubleshooting

‚ùå "Datasheets are stale" error

Cause: Generated datasheets are older than 10 hours.
Solution: Run "Generate Datasheets" again RIGHT BEFORE Transfer.

‚ùå Folder not recognized

Cause: Folder name doesn't match Ïù¥Î¶Ñ_Category format.
Solution: Rename folder to correct format (e.g., ÍπÄÎØºÏòÅ_Quest).

‚ùå Excel file locked/corrupted

Cause: Microsoft Security Label still on file.
Solution: Open file ‚Üí File ‚Üí Info ‚Üí Remove sensitivity label ‚Üí Save.

‚ùå Transfer finds no folders

Cause: QAfolderOLD is empty or folders have wrong names.
Solution: Move QAfolder contents to QAfolderOLD first.

‚ùå Wrong language datasheet copied

Cause: Tester not in languageTOtester_list.txt.
Solution: Add tester name under correct language section (ENG or ZHO-CN).
