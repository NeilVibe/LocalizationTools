==================================
QAfolderOLD - Place OLD QA Files Here
==================================

This folder is for EXISTING QA work from testers.

Expected structure:
  QAfolderOLD/
    {TesterName}_{Category}/
      {TesterName}_{Category}.xlsx
      screenshot1.png
      screenshot2.png
      ...

Example:
  QAfolderOLD/
    John_Quest/
      John_Quest.xlsx
      quest_bug1.png
    Mary_Item/
      Mary_Item.xlsx
      item_issue.png

Valid categories:
  - Quest
  - Knowledge
  - Item
  - Region
  - System
  - Character
  - Skill
  - Help
  - Gimmick

After placing files here, run "Transfer QA Files" in the GUI.
