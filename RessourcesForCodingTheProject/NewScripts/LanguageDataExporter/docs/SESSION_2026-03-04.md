### `config.py` Changes
```python
def _rebuild_paths():
    """Shared helper — called by both update_branch() and update_drive()."""
    global LOC_FOLDER, EXPORT_FOLDER, VOICE_RECORDING_FOLDER
    LOC_FOLDER = _build_path(r"F:\perforce\cd\mainline\resource\GameData\stringtable\loc")
    EXPORT_FOLDER = _build_path(r"F:\perforce\cd\mainline\resource\GameData\stringtable\export__")
    VOICE_RECORDING_FOLDER = _build_path(r"F:\perforce\cd\mainline\resource\editordata\VoiceRecordingSheet__")

def update_drive(new_drive: str):
    global _DRIVE_LETTER
    clean = new_drive.strip().rstrip(':').upper()   # sanitize: strip spaces+colon, take first char
    if not clean or not clean[0].isalpha():
        logger.warning(f"Invalid drive letter ignored: {new_drive!r}")
        return
    _DRIVE_LETTER = clean[0]
    _rebuild_paths()
    _SETTINGS['drive_letter'] = _DRIVE_LETTER
    _save_settings(_SETTINGS)

def get_drive() -> str:
    return _DRIVE_LETTER
```

## Build 032 — Include Script Toggle

### Feature
Toggle in the Actions section to include or exclude Script/Dialog content from Excel generation.

**OFF (default):** ENG and ZHO-CN exclude `{Sequencer, AIDialog, QuestDialog, NarrationDialog}` — these are voiced languages that handle dialog separately.

**ON:** All languages get full languagedata (no exclusions applied to any language).

### Terminology Note
- LanguageDataExporter calls this content `DIALOG_SEQUENCER_EXCLUSION`
- QuickTranslate calls the same content `SCRIPT_CATEGORIES = {"Sequencer", "Dialog"}`
- Same source files, different naming conventions. "Include Script" = include full languagedata.

### Implementation
```python
# In __init__:
self.include_script_var = tk.BooleanVar(value=False)

# Capture on main thread BEFORE spawning background thread:
include_script = self.include_script_var.get()

# Script exclusion logic in background thread:
if not include_script and lang_code.lower() in LANGUAGES_WITH_DIALOG_EXCLUSION:
    excluded_categories = DIALOG_SEQUENCER_EXCLUSION
```

### `_set_buttons_enabled()` Updated
Disables `btn_excel`, `btn_report`, AND `chk_include_script` during generation.

### FileName Column

New column showing which `.loc.xml` file each StringID came from.

**Column position:** Between `Category` and `StringID` (second-to-last).
**Width:** 25.

**New function in `category_mapper.py`:**
```python
def build_stringid_filename_index(export_folder: Path) -> Dict[str, str]:
    filename_index: Dict[str, str] = {}
    for xml_file in export_folder.rglob("*.loc.xml"):
        stem = xml_file.name[:-8] if xml_file.name.endswith(".loc.xml") else xml_file.stem
        string_ids = parse_export_file(xml_file)
        for string_id in string_ids:
            if string_id not in filename_index:
                filename_index[string_id] = stem
    return filename_index
```

Exported from `exporter/__init__.py` and wired into `gui/app.py` alongside `build_stringid_category_index`.

## Key Architecture Notes

### Path Resolution Chain
```
settings.json (drive_letter + branch)
    ↓ _load_settings() at import time
    ↓ _build_path() applies drive + branch substitution
LOC_FOLDER / EXPORT_FOLDER / VOICE_RECORDING_FOLDER (module globals)
    ↓ _rebuild_paths() called by update_branch() and update_drive()
GUI path labels → _check_paths() → green OK / red NOT FOUND
```

### Threading Pattern
All main-thread state must be captured as locals BEFORE spawning background thread:
```python
loc_folder = config.LOC_FOLDER          # capture now
export_folder = config.EXPORT_FOLDER    # capture now
include_script = self.include_script_var.get()   # capture now
selected_langs = self._get_selected_languages()  # capture now

def _run():
    # use loc_folder, export_folder, include_script, selected_langs (locals)
    # NEVER access self.include_script_var from background thread
    ...
    root.after(0, lambda: self._set_status("Done"))  # tkinter updates via after()

threading.Thread(target=_run, daemon=True).start()
```

### Category Mapper: Two Independent Indexes Built Simultaneously
```python
category_index = build_stringid_category_index(export_folder, cluster_config)
filename_index = build_stringid_filename_index(export_folder)
```
Both scan the same EXPORT folder with `rglob("*.loc.xml")`. Could be merged into one pass for performance, but kept separate for clarity.
