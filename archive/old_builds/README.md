# Updates Directory

This directory stores built Electron app installers for self-hosted distribution.

## Structure

```
updates/
├── LocaNext Setup 1.0.0.exe  # Windows installer
├── latest.yml                 # Update manifest (auto-generated by electron-builder)
└── README.md                  # This file
```

## How to Add a New Version

1. Build the app:
```bash
cd locaNext
npm run build
```

2. Copy files here:
```bash
cp dist-electron/LocaNext\ Setup\ 1.0.0.exe ../updates/
cp dist-electron/latest.yml ../updates/
```

3. Edit `latest.yml` to point to your server:
```yaml
url: http://yourserver.com/updates/download/LocaNext-Setup-1.0.0.exe
```

4. Users can download from:
```
GET /updates/download/LocaNext-Setup-1.0.0.exe
```

## API Endpoints

The FastAPI server serves these files:

- `GET /updates/latest.yml` - Update manifest
- `GET /updates/download/{filename}` - Download installer
- `GET /updates/version` - Get current version info

## Security

Only `.exe`, `.nupkg`, `.zip`, and `.yml` files can be downloaded (enforced by API).
