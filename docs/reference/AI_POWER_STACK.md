## Qwen3-VL Vision Model — Discovery & Usage

### What We Discovered (2026-03-19)
Qwen3-VL:8B is loaded in Ollama (6.1GB) and configured in OpenViking, but had **NEVER been called** — zero vision requests in logs. All 93 OpenViking documents are text files.

### Test Result
Fed it a landing page screenshot. It correctly:
- Read Korean text ("오픈소스 · 무료 · 엔터프라이즈급")
- Identified it as a localization/game development platform
- Recognized language node labels (KR, EN, JP, TR, DE, CN)
- Understood the product purpose from visual context alone

### How to Use Qwen3-VL

**Via Ollama API (direct):**
```python
import requests, base64

with open('screenshot.png', 'rb') as f:
    img_b64 = base64.b64encode(f.read()).decode()

r = requests.post('http://localhost:11434/api/chat', json={
    'model': 'qwen3-vl:8b',
    'messages': [{
        'role': 'user',
        'content': 'Describe this image in detail.',
        'images': [img_b64]
    }],
    'stream': False
}, timeout=300)

print(r.json()['message']['content'])
```

**Via OpenViking (for indexed documents):**
Configured at `~/.openviking/ov.conf` line 27:
```json
"model": "ollama/qwen3-vl:8b"
```
Index images/screenshots into Viking → they become searchable via semantic queries.
