# LocaNext 프로젝트 개요

---

## 프로젝트 소개

**LocaNext**는 번역 및 로컬라이제이션 작업을 위한 **통합 데스크톱 플랫폼**입니다.

---

## 핵심 가치

### 1. 통합 플랫폼
- 10-20개 이상의 도구를 하나의 앱에서 제공
- 여러 프로그램 설치 불필요

### 2. 고성능 처리
- 사용자 PC에서 직접 처리
- PostgreSQL 데이터베이스 (31K entries/sec)
- 빠른 처리 속도

### 3. 전문가용 품질
- 기업 환경에 적합한 보안
- 912개 테스트로 검증됨

## 시스템 구조

### 사용자 PC (데스크톱 앱)

| 구성요소 | 기술 | 용도 |
|----------|------|------|
| LocaNext.exe | Electron | 사용자 인터페이스 |
| 백엔드 서버 | Python + FastAPI | 처리 엔진 |

### 중앙 서버

| 구성요소 | 기술 | 용도 |
|----------|------|------|
| API 서버 | FastAPI | 데이터 처리 |
| **데이터베이스** | **PostgreSQL** | 모든 데이터 저장 |
| 연결 풀링 | PgBouncer | 1000 동시 연결 |
| 관리자 대시보드 | SvelteKit | 모니터링 |

### 통신 방식
- HTTPS (REST API) - 데이터 교환
- WSS (WebSocket) - 실시간 업데이트
