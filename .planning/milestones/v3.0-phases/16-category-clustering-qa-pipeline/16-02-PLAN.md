<objective>
Add inline QA indicators to grid rows and enhance the QA panel with per-cell dismiss and summary counts.

Purpose: Translators need to see QA issues (term consistency, line uniformity) directly in the grid without opening a side panel, and they need to dismiss false positives per-cell to prevent alert fatigue. The QA backend (Term Check with Aho-Corasick, Line Check, resolve/dismiss) already exists from v1.0 Phase 5.1 -- this plan adds inline display, dismiss UX, and enhanced summary.

Output: QAInlineBadge component, enhanced QAMenuPanel, inline indicators in VirtualGrid, integration tests.
</objective>
