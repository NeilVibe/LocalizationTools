From server/tools/ldm/schemas/codex.py (created in Plan 01):
```python
class CodexEntity(BaseModel):
    entity_type: str          # "character", "item", "skill", "region", "gimmick", "knowledge"
    strkey: str               # STR_CHAR_0001
    name: str
    description: Optional[str] = None
    knowledge_key: Optional[str] = None
    image_texture: Optional[str] = None  # UITextureName for /mapdata/thumbnail/{texture_name}
    audio_key: Optional[str] = None      # StrKey for /mapdata/audio/stream/{strkey}
    source_file: str
    attributes: dict = {}     # Type-specific: Grade, Job, Race, Gender, Age, etc.
    related_entities: List[str] = []

class CodexSearchResult(BaseModel):
    entity: CodexEntity
    similarity: float
    match_type: str  # "semantic" or "exact"

class CodexSearchResponse(BaseModel):
    results: List[CodexSearchResult]
    count: int
    search_time_ms: float

class CodexListResponse(BaseModel):
    entities: List[CodexEntity]
    entity_type: str
    count: int
```

- GET /api/ldm/codex/search?q={query}&entity_type={type}&limit={n} -> CodexSearchResponse
- GET /api/ldm/codex/entity/{entity_type}/{strkey} -> CodexEntity
- GET /api/ldm/codex/list/{entity_type} -> CodexListResponse
- GET /api/ldm/codex/types -> dict[str, int]

Media endpoints (existing):
- GET /api/ldm/mapdata/thumbnail/{texture_name} -> PNG image
- GET /api/ldm/mapdata/audio/stream/{string_id} -> WAV audio
