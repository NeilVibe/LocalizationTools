From server/tools/ldm/schemas/worldmap.py (created by Plan 20-01):
```python
class MapNode(BaseModel):
    strkey: str               # FNODE_0001
    knowledge_key: str        # KNOW_REGION_0001
    name: str                 # Region name from Codex
    description: Optional[str] = None
    region_type: str          # Main, Sub, Dungeon, Town, Fortress, Wilderness
    x: float                  # WorldPosition X (range 500-5000)
    z: float                  # WorldPosition Z (range 500-5000)
    npcs: List[str] = []      # Character names in this region
    entity_type_counts: dict = {}  # {"character": 3, "item": 5}

class MapRoute(BaseModel):
    from_node: str            # FNODE_0001
    to_node: str              # FNODE_0002
    waypoints: List[dict] = []  # [{"x": float, "z": float}]

class WorldMapData(BaseModel):
    nodes: List[MapNode]
    routes: List[MapRoute]
    bounds: dict = {}         # {"min_x", "max_x", "min_z", "max_z"}
```
