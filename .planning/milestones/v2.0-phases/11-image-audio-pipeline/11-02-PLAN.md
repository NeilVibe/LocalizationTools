    - test_thumbnail_cache_header: response includes Cache-Control header with max-age

  <action>
    **Add two endpoints to `server/tools/ldm/routes/mapdata.py`:**

    1. `GET /mapdata/thumbnail/{texture_name}`:
       - Import `get_media_converter` from media_converter module
       - Import `Response` from fastapi.responses (already available via fastapi)
       - Look up `texture_name.lower()` in `service._dds_index`
       - If not found, raise HTTPException(404, f"Texture '{texture_name}' not found")
       - Convert DDS path to WSL path via `convert_to_wsl_path(str(dds_path))`
       - Call `converter.convert_dds_to_png(Path(wsl_path))`
       - If None, raise HTTPException(500, f"Failed to convert '{texture_name}'")
       - Return `Response(content=png_bytes, media_type="image/png", headers={"Cache-Control": "public, max-age=86400"})`
       - Use `asyncio.to_thread` for the conversion call to avoid blocking event loop

    2. `GET /mapdata/audio/stream/{string_id}`:
       - Look up audio context via `service.get_audio_context(string_id)`
       - If None, raise HTTPException(404, f"No audio for '{string_id}'")
       - Convert WEM path to WSL path
       - Call `converter.convert_wem_to_wav(Path(wsl_path))` via `asyncio.to_thread`
       - If None, raise HTTPException(500, "Audio conversion failed")
       - Return `FileResponse(wav_path, media_type="audio/wav")`

    Both endpoints require auth via `Depends(get_current_active_user_async)`.
