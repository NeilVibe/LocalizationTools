    ```xml
    <?xml version="1.0" encoding="UTF-8"?>

    ```

    All XML files must use raw `<br/>` tags inside attribute values (NOT escaped). The XML parser handles this via CDATA or attribute value handling.

    IMPORTANT: LocStr XML uses `<br/>` as literal text in attribute values, which means in the XML source they appear as `&lt;br/&gt;` (XML-escaped) within attribute strings.
