    ```python
        """3-Tier AC Context Search for Korean TM text.

        Tier 1: Whole AC match — source contains a whole TM entry as substring
        Tier 2: Line AC match — source contains a line TM entry as substring
        Tier 3: Char n-gram Jaccard fuzzy — n={2,3,4,5}, space-stripped Korean, >= 62%

        Constructor takes indexes dict from TMIndexer.load_indexes() (which now includes
        whole_automaton and line_automaton).
        """

        CONTEXT_THRESHOLD = 0.62  # 62% Jaccard threshold for fuzzy tier

            self.whole_automaton = indexes.get("whole_automaton")
            self.line_automaton = indexes.get("line_automaton")
            self.whole_lookup = indexes.get("whole_lookup", {})
            self.line_lookup = indexes.get("line_lookup", {})

        def search(self, source_text: str, max_results: int = 10) -> Dict[str, Any]:
            """Run 3-tier cascade on source_text. Returns dict with tiers and results."""
            # Normalize source

            # Combine: tier-1 first, tier-2 next, tier-3 last. Within each tier, sort by score desc.
            # Return {"results": [...], "tier_counts": {"whole": N, "line": N, "fuzzy": N}, "total": N}
    ```

    Each result dict: `{"source_text": str, "target_text": str, "entry_id": int, "score": float, "tier": str, "match_type": str}`

    For Tier 1 (whole AC):
    - `normalize_for_hash(source_text)` then iterate `whole_automaton.iter(normalized)`
    - Each hit: look up the matched key in `whole_lookup` to get target_text
    - Score = 1.0, tier = "whole"
    - Deduplicate by entry_id

    For Tier 2 (line AC):
    - `normalize_for_hash(source_text)` then iterate `line_automaton.iter(normalized)`
    - Each hit: look up matched key in `line_lookup`
    - Score = 1.0, tier = "line"
    - Deduplicate by entry_id (skip if already found in Tier 1)

    For Tier 3 (fuzzy Jaccard):
    - Space-strip source: `source_stripped = normalize_for_hash(source_text).replace(" ", "")`
    - Compute multi-n-gram set: union of n-grams for n in {2,3,4,5}
    - For each key in whole_lookup:
      - Space-strip key: `key_stripped = key.replace(" ", "")`
      - Compute multi-n-gram set for key
      - Jaccard = |intersection| / |union|
      - If Jaccard >= 0.62 (CONTEXT_THRESHOLD), add to results
    - Skip entries already found in Tier 1 or 2
    - Sort by Jaccard descending

    Helper methods:
    - `_get_multi_ngrams(text: str, ns: tuple = (2,3,4,5)) -> set`: returns union of character n-grams for each n
    - `_strip_spaces_korean(text: str) -> str`: remove all spaces (Korean spacing is inconsistent)

    IMPORTANT: For Tier 3, handle whole_lookup entries that have "variations" key — iterate variations list to get individual entry_ids.
