      - name_translated: find _NAME StringId via mega.entity_stringids("region", strkey) + mega.get_translation
      - Resolve faction_name: find parent FactionEntry by iterating mega.faction_by_strkey where strkey in entry.parent_strkey chain OR node_strkeys contains strkey
      - Resolve faction_group_name: from FactionEntry.group_strkey -> FactionGroupEntry.group_name

       - Build FactionGroupNode -> FactionNode -> FactionNodeItem hierarchy
       - region_count on FactionGroupNode = sum of all regions across factions
       - has_position = entry.world_position is not None
