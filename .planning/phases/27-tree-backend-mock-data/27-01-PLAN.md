    **Step 2: Create gamedata_tree_service.py**:

    New file `server/tools/ldm/services/gamedata_tree_service.py` with class `GameDataTreeService`:

    - Constructor: `__init__(self, base_dir: Path | str)` storing `self.base_dir = Path(base_dir).resolve()`
    - `_validate_path(self, path: str) -> Path` — reuse same logic as GameDataBrowseService._validate_path
    - `parse_file(self, xml_path: str, max_depth: int = -1) -> GameDataTreeResponse`:
      1. Validate and resolve path
      2. Parse XML with `etree.parse(str(resolved))`, fallback to `etree.fromstring()` with `recover=True` parser
      3. Get root element, iterate direct children with `list(root)`
      4. For each direct child element, call `_build_subtree(element, depth=0, max_depth=max_depth)`
      5. After building initial tree, call `_resolve_parent_references(roots)` to restructure flat ParentId-based elements into nested hierarchy
      6. Return GameDataTreeResponse with roots, file_path, entity_type (tag of first child), node_count

    - `_build_subtree(self, element, depth, max_depth, id_prefix) -> TreeNode`:
      1. Create TreeNode with tag=element.tag, attributes=dict(element.attrib)
      2. Generate node_id from id_prefix + index
      3. Set editable_attrs from EDITABLE_ATTRS.get(element.tag, [])
      4. If max_depth != -1 and depth >= max_depth, return without children
      5. For each child element in `list(element)`, recursively call _build_subtree
      6. Return the TreeNode

    - `_resolve_parent_references(self, roots: List[TreeNode]) -> List[TreeNode]`:
      THE KEY METHOD -- handles ParentId/ParentNodeId resolution:
      1. For each root, collect all children into a flat lookup dict keyed by their identifying attribute (NodeId, Id, Key)
      2. Check if any child has attribute "ParentNodeId" or "ParentId" (case-insensitive check: try ParentNodeId, ParentId, parentnodeid, parentid)
      3. If ParentId/ParentNodeId references exist:
         a. Build parent_map: {parent_id_value -> [child_nodes]}
         b. Nodes with ParentNodeId="0" or ParentId="0" stay as direct children of root
         c. Nodes with ParentNodeId=N become children of the node whose NodeId/Id=N
         d. Remove re-parented nodes from root's direct children list
         e. Set parent_id field on each moved node
      4. Return modified roots

    - `parse_folder(self, folder_path: str, max_depth: int = -1) -> FolderTreeDataResponse`:
      1. Validate path, ensure it's a directory
      2. Recursively find all .xml files using Path.rglob("*.xml")
      3. For each XML file, call parse_file()
      4. Return FolderTreeDataResponse with all file trees, base_path, total_nodes

    **Step 2: Add POST /gamedata/tree endpoint** (after existing /gamedata/rows endpoint):

    ```python
    @router.post("/gamedata/tree", response_model=GameDataTreeResponse)
    async def get_gamedata_tree(
        request: TreeRequest,
        current_user: dict = Depends(get_current_active_user_async),
    ):
        """Parse an XML gamedata file and return hierarchical tree with nested nodes.

        Handles both XML-nested hierarchies (GimmickGroup > GimmickInfo) and
        reference-based hierarchies (SkillNode ParentNodeId).

        Args:
            request: Contains path to XML file and optional max_depth (-1 = unlimited).

        Returns:
            GameDataTreeResponse with nested TreeNode roots.
        """
        base_dir = _get_base_dir()
        try:
            svc = GameDataTreeService(base_dir=base_dir)
            result = svc.parse_file(request.path, max_depth=request.max_depth)
        except ValueError as e:
            logger.warning(f"[GameData API] Path traversal attempt: {e}")
            raise HTTPException(status_code=403, detail=str(e))
        except FileNotFoundError:
            raise HTTPException(status_code=404, detail=f"File not found: {request.path}")
        except XMLSyntaxError as e:
            logger.warning(f"[GameData API] Malformed XML: {request.path}: {e}")
            raise HTTPException(status_code=422, detail=f"Malformed XML: {e}")
        except Exception as e:
            logger.error(f"[GameData API] get_gamedata_tree failed: {e}")
            raise HTTPException(status_code=500, detail="Internal server error")
        return result
    ```
