For each of the 5 page screenshots, send to Qwen3-VL for visual review:

1. For each screenshot (files, gamedev, codex, map, tm):
   a. Base64-encode the WebP file
   b. POST to http://localhost:11434/api/generate with:
      - model: "qwen3-vl:8b"
      - images: [base64_encoded_image]
      - prompt: "You are a UI/UX expert reviewing a localization tool interface. Rate this page 1-10 on visual quality, data presentation, layout, and professionalism. List any critical issues (broken layouts, missing data, misaligned elements, empty states that should have data, poor contrast, truncated text). Format: SCORE: X/10 then CRITICAL ISSUES: bulleted list then MINOR ISSUES: bulleted list."
      - stream: false
   c. Parse the response to extract score and issues
   d. Log score and issues

2. Compile all results into a structured JSON file at `.planning/phases/79-visual-audit/79-01-audit-results.json`:

```json
  }
}
```

3. Print summary table: Page | Score | Critical Issues Count | Pass (7+)?
