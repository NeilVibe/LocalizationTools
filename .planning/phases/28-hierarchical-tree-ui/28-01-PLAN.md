```python
class TreeNode(BaseModel):
    node_id: str      # unique ID (e.g., "r0_c2")
    tag: str           # XML element name (e.g., "SkillTreeInfo", "SkillNode")
    attributes: Dict[str, Any] = {}
    children: List[TreeNode] = []
    parent_id: Optional[str] = None
    editable_attrs: List[str] = []

class GameDataTreeResponse(BaseModel):
    roots: List[TreeNode]
    file_path: str
    entity_type: str    # tag name of root's children
    node_count: int

class FolderTreeDataResponse(BaseModel):
    files: List[GameDataTreeResponse]
    base_path: str
    total_nodes: int
```

**Props (via $props()):**
- `filePath: string | null` -- when set, fetch tree for this single file via POST /api/ldm/gamedata/tree
- `folderPath: string | null` -- when set via "Load All", fetch folder tree via POST /api/ldm/gamedata/tree/folder
- `onNodeSelect: (node) => void` -- callback when a tree node is clicked (for detail panel in Plan 02)

**State (Svelte 5 $state):**
- `treeData = $state(null)` -- array of GameDataTreeResponse objects (one per file, or one for single file)
- `expandedNodes = $state(new Set())` -- set of node_id strings for expanded nodes
- `selectedNodeId = $state(null)` -- currently selected node's node_id
- `loading = $state(false)` -- loading indicator
- `error = $state(null)` -- error message string

**API integration:**
- `loadFileTree(path)`: POST to `${API_BASE}/api/ldm/gamedata/tree` with body `{ path }`, parse response as GameDataTreeResponse, store in treeData as single-element array. Auto-expand root level by adding all root node_ids to expandedNodes.
- `loadFolderTree(path)`: POST to `${API_BASE}/api/ldm/gamedata/tree/folder` with body `{ path }`, parse response as FolderTreeDataResponse. Each file becomes a collapsible group in the tree. Store files array in treeData.
- Use `getAuthHeaders()` and `getApiBase()` from `$lib/utils/api.js`.
- Use `logger.apiCall()`, `logger.success()`, `logger.error()` from `$lib/utils/logger.js`.

**$effect for reactive loading:**
```javascript
$effect(() => {
  if (filePath) loadFileTree(filePath);
});
```

**Tree rendering (Svelte 5 {#snippet}):**
Define a recursive `{#snippet renderNode(node, depth)}` following FileExplorerTree's `renderFolder` pattern:

- Each node row: `<button class="tree-node" style="padding-left: {depth * 20 + 8}px">`
  - Chevron icon: `ChevronDown` if expanded, `ChevronRight` if collapsed (only if node.children.length > 0)
  - Entity type icon: use a `getNodeIcon(tag)` function returning Carbon icon component based on tag name:
    - SkillTreeInfo, SkillInfo, SkillNode -> `Skill` (use `Fleet` icon from carbon-icons-svelte)
    - ItemInfo -> `ShoppingCatalog`
    - CharacterInfo -> `UserAvatar`
    - GimmickGroupInfo, GimmickInfo -> `GameWireless`
    - KnowledgeInfo -> `Book`
    - QuestInfo -> `Task`
    - RegionInfo, SceneObjectData -> `Map`
    - SealDataInfo -> `Locked`
    - FactionGroup -> `GroupPresentation`
    - NodeWaypointInfo -> `LocationStar`
    - Default (unknown tags) -> `DataStructured`
  - Primary label: derive from EDITABLE_ATTRS -- show `node.attributes[EDITABLE_ATTRS[node.tag][0]]` if available, otherwise show `node.attributes.Key || node.attributes.NodeId || node.tag`
  - Children count badge: if node.children.length > 0 show `<span class="count-badge">{node.children.length}</span>`

- Click on chevron area or double-click toggles expand/collapse (same Set mutation pattern as FileExplorerTree)
- Single click selects the node: set selectedNodeId, call onNodeSelect(node)
- Selected node gets class `selected` with styling: `background: var(--cds-layer-selected-01); border-left: 3px solid var(--cds-link-01);`

**Keyboard navigation:**
- ArrowDown/ArrowUp: move selection to next/previous visible node
- ArrowRight: expand selected node (if collapsed) or move to first child
- ArrowLeft: collapse selected node (if expanded) or move to parent
- Enter: toggle expand/collapse
- Space: select node (trigger onNodeSelect)
