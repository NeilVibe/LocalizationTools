Create 5 Svelte 5 components using Runes ($state, $derived, $effect, $props). All components go in `locaNext/src/lib/components/common/`. Use Carbon components (Button, SkeletonText) where available. Reference the design tokens from Task 1.

**PageHeader.svelte** (~40 lines):
- Props: `icon` (Carbon icon component, optional), `title` (string), children snippet (Svelte 5 `{@render children?.()}` for action slot)
- Layout: flex row, space-between, icon + h1 title on left, action slot on right
- Styling: uses `--page-header-padding`, `--page-header-height`, `var(--cds-layer-01)` background, `var(--panel-border)` bottom border
- Typography: h1 at 1.125rem/600 weight, `var(--cds-text-01)` color
- Icon + title gap: 0.5rem

**SkeletonCard.svelte** (~50 lines):
- Props: `count` (number, default 6) -- how many skeleton cards to render
- Layout: Each card is a flex row matching Codex entity-card dimensions: 48x48 image placeholder + text lines (wide 70%, tag 40px/16px tall with border-radius 8px, narrow 85%)
- Uses `var(--card-padding)`, `var(--card-gap)`, `var(--card-radius)`, `var(--skeleton-bg)`
- CSS-only shimmer animation: opacity 0.4 to 0.8 at 1.5s ease-in-out infinite
- Background: `var(--cds-layer-01)`, border: `1px solid var(--cds-border-subtle-01)`
- Use `{#each Array(count) as _, i (i)}` with key

**EmptyState.svelte** (~30 lines):
- Props: `icon` (Carbon icon component, optional), `headline` (string), `description` (string, optional), `ctaLabel` (string, optional), `onaction` (callback, optional)
- Layout: centered flex column with 1rem gaps
- Icon: renders at size 48, `var(--cds-text-03)` color
- Headline: 1rem font-size, `var(--cds-text-01)`, font-weight 500
- Description: 0.875rem, `var(--cds-text-03)`
- CTA: Carbon `Button` kind="tertiary" size="small", only renders if ctaLabel provided, calls onaction on click

**ErrorState.svelte** (~25 lines):
- Props: `message` (string, default "Something went wrong"), `onretry` (callback, optional)
- Uses Carbon `WarningAlt` icon at size 32 in `var(--cds-support-01)` (red)
- Message in `var(--cds-text-01)` at 0.875rem
- Retry button: Carbon `Button` kind="ghost" size="small" with icon `Restart`, only renders if onretry provided
- Layout: centered flex column, 0.75rem gap, padding 2rem

**InfiniteScroll.svelte** (~30 lines):
- Props: `onloadmore` (callback), `loading` (boolean, default false), `hasMore` (boolean, default true)
- Uses `$effect` to set up IntersectionObserver on a sentinel div
- `rootMargin: '200px'` for prefetch feel
- Observer disconnects via `return () => observer.disconnect()` in $effect cleanup
- When `entry.isIntersecting` and not loading and hasMore, calls `onloadmore()`
- $effect re-runs when `hasMore` or `loading` change (they are reactive props)
- Sentinel div: `bind:this={sentinel}`, `aria-hidden="true"`, min-height 1px
- If loading, renders a `<slot name="loading" />` below sentinel (use Svelte 5 snippet: `{#if loading}{@render loadingSlot?.()}{/if}` where loadingSlot is from $props)
- If !hasMore, renders nothing (sentinel hidden)
