    3. def _find_similar_segments(self, source_text: str, top_k: int = 3) -> list[dict]:
       - Import get_embedding_engine from server.tools.shared
       - Try to get the engine via get_embedding_engine()
       - Encode source_text: engine.encode([source_text], normalize=True)
       - Use the TM FAISS index if available (check TMSearcher pattern from indexing/searcher.py)
       - If no FAISS index or engine not available, try tm_repo.search_similar() as fallback
       - If ALL embedding/search fails (no TM loaded, no index), return [] gracefully (do NOT crash)
       - Return top_k results as [{"source": str, "target": str, "similarity": float}]
       - Wrap entire method in try/except to ensure robustness -- embedding search is enrichment, not required

    4. def _blend_confidence(self, llm_confidence: float, similar_segments: list[dict]) -> float:
       - If similar_segments is empty: return clamped llm_confidence (no embedding data to blend)
       - max_embedding_sim = max(s["similarity"] for s in similar_segments)
       - blended = BLEND_WEIGHT_EMBEDDING * max_embedding_sim + BLEND_WEIGHT_LLM * llm_confidence
       - Return min(1.0, max(0.0, blended))

    5. async def generate_suggestions(self, string_id, source_text, target_lang, entity_type, surrounding_context) -> dict:
       - Check cache first (key = f"{string_id}:{target_lang}:{hashlib.md5(source_text.encode()).hexdigest()[:8]}")
       - Step 1: Find similar segments via _find_similar_segments(source_text) -- per AISUG-02
       - Step 2: Build prompt via _build_prompt() including entity_type, surrounding_context (max 4 segments), AND similar_segments (top 3 with source/target/similarity)
       - Step 3: POST to Ollama with format=AISuggestionResponse.model_json_schema(), temperature=0.7, num_predict=500
       - Step 4: Parse response, blend confidence via _blend_confidence(llm_confidence, similar_segments) for each suggestion
       - Cache and return {"suggestions": [...], "status": "generated"}
       - ConnectError/TimeoutException -> {"suggestions": [], "status": "unavailable"}
       - Other exceptions -> {"suggestions": [], "status": "error"}

    6. def _build_prompt(self, source_text, target_lang, entity_type, surrounding_context, similar_segments) -> str:
       - Professional game localization translator persona
       - Include source_text (truncated to 500 chars), target_lang, entity_type
       - Include similar_segments as translation examples (top 3: "Source: X -> Target: Y (similarity: N%)")
       - Include surrounding context lines (max 4, source -> target pairs)
       - Request 3 suggestions with confidence and reasoning
       - Truncate all inputs to prevent prompt injection

    7. Singleton pattern: get_ai_suggestion_service() identical to get_ai_summary_service()
