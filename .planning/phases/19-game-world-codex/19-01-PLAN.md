       - `_build_search_index()`: concatenate entity name + description text, encode via get_embedding_engine(), build FAISS index via FAISSManager.build_index(), store aligned key list [(type, strkey), ...]
       - `initialize()`: call _scan_entities, _resolve_cross_refs, _find_related_entities, _build_search_index. Lazy-called on first request.
       - `search(query: str, entity_type: Optional[str], limit: int = 20) -> CodexSearchResponse`: encode query, FAISSManager.search(), filter by entity_type if provided, return results with timing
       - `get_entity(entity_type: str, strkey: str) -> Optional[CodexEntity]`: direct registry lookup
       - `list_entities(entity_type: str) -> CodexListResponse`: return all entities of type
       - Set audio_key = entity.strkey for all entities (audio is keyed by StrKey)
