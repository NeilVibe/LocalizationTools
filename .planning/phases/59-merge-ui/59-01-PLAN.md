- Parse SSE response manually: maintain buffer, split by newlines, handle event: and data: prefixes
- SSE event handling:
  - `progress`: append message text to progressMessages array, update progressPercent estimate
  - `log`: append formatted log message

  - `error`: show error in InlineNotification, set executing=false

- Carbon `ProgressBar` showing estimated progress (count processed files vs previewResult.files_processed)
