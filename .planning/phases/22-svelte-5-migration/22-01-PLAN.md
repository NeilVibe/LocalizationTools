    1. Remove `import { createEventDispatcher } from "svelte"` and `const dispatch = createEventDispatcher()`
    2. Add callback props to the `$props()` destructuring: `let { ..., onEventName = undefined } = $props()`
    3. Replace every `dispatch('eventName', data)` with `onEventName?.(data)`
    4. If the component also CONSUMES child events via `on:eventName`, convert those too: `on:eventName={handler}` becomes `onEventName={handler}`

    **VirtualGrid dispatch map (20 calls):**
    - `dispatch('inlineEditStart', {...})` -> `onInlineEditStart?.({...})`
    - `dispatch('rowUpdate', {...})` -> `onRowUpdate?.({...})`
    - `dispatch('rowSelect', {...})` -> `onRowSelect?.({...})`
    - `dispatch('confirmTranslation', {...})` -> `onConfirmTranslation?.({...})`
    - `dispatch('dismissQA', {...})` -> `onDismissQA?.({...})`
    - `dispatch('runQA', {...})` -> `onRunQA?.({...})`
    - `dispatch('addToTM', {...})` -> `onAddToTM?.({...})`
    - All remaining dispatch calls follow the same pattern: `dispatch('X', data)` -> `onX?.(data)`
    - Also convert `on:select={handleFilterChange}` on the CategoryFilter Carbon Dropdown to keep it Carbon-exempt if it's a Carbon component, OR convert if it's the custom CategoryFilter component

    **RightPanel:** Remove dispatcher, add `onApplyTM`, `onApplySuggestion`, `onDismissQA`, `onOpenEditModal` as callback props.
