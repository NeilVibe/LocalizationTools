For each image in the media results, replace the direct `<img>` with:

```svelte
<div class="image-container">
  {#if imageLoadStates.get(img.path || img.url || idx) !== 'loaded'}
    <div class="image-shimmer"></div>
  {/if}
  <img
    src="{API_BASE}/api/ldm/gamedata/media/image?path={encodeURIComponent(img.path || img.url)}"
    alt={img.name || 'Media'}
    class="media-image"
    class:image-reveal={imageLoadStates.get(img.path || img.url || idx) === 'loaded'}
    class:image-hidden={imageLoadStates.get(img.path || img.url || idx) !== 'loaded'}
    loading="lazy"
    onload={() => {
      const next = new Map(imageLoadStates);
      next.set(img.path || img.url || idx, 'loaded');
      imageLoadStates = next;
    }}
    onerror={() => {
      const next = new Map(imageLoadStates);
      next.set(img.path || img.url || idx, 'error');
      imageLoadStates = next;
    }}
  />
</div>
```

NOTE: Adapt the `src` and key expressions to match the actual image data structure in the existing code. Read the file first to find the exact field names.
