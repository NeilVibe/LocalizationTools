Per GRAFT-09: Create NEW file `locaNext/src/lib/components/ldm/LoadingScreen.svelte`.

This is a Svelte 5 Runes component. Requirements:
- Centered vertically and horizontally in parent container
- Progress bar with smooth CSS transition (width animation)
- Percentage text below bar
- Optional message text (e.g., "Loading XML files...")
- Use Carbon Design token colors for consistency
- Professional, industry-grade appearance (not shimmer)

```svelte
<script>
  /**
   * LoadingScreen.svelte - Professional loading indicator
   * Replaces shimmer skeletons with centered progress bar + percentage.
   */

  let {
    progress = 0,       // 0-100
    message = 'Loading...',
    showPercentage = true
  } = $props();

  let displayProgress = $derived(Math.min(100, Math.max(0, Math.round(progress))));
</script>

<div class="loading-screen">
  <div class="loading-content">
    <div class="loading-icon">
      <!-- Simple pulsing dot animation -->
      <div class="pulse-ring"></div>
      <div class="pulse-dot"></div>
    </div>

    {#if message}
      <p class="loading-message">{message}</p>
    {/if}

    <div class="progress-container">
      <div class="progress-track">
        <div
          class="progress-fill"
          style="width: {displayProgress}%"
        ></div>
      </div>
      {#if showPercentage}
        <span class="progress-text">{displayProgress}%</span>
      {/if}
    </div>
  </div>
</div>
```

CSS requirements:
- `.loading-screen`: `display: flex; align-items: center; justify-content: center; flex: 1; min-height: 200px;`
- `.loading-content`: `text-align: center; max-width: 320px; width: 100%;`
- `.progress-track`: `height: 4px; background: var(--cds-layer-02); border-radius: 2px; overflow: hidden;`
- `.progress-fill`: `height: 100%; background: var(--cds-interactive); border-radius: 2px; transition: width 0.3s ease;`
- `.progress-text`: `font-size: 0.75rem; color: var(--cds-text-02); margin-top: 0.5rem;`
- `.loading-message`: `font-size: 0.875rem; color: var(--cds-text-01); margin-bottom: 1rem;`
- Pulse animation: `@keyframes pulse { 0%, 100% { transform: scale(1); opacity: 0.8; } 50% { transform: scale(1.2); opacity: 1; } }` on `.pulse-dot` with `animation: pulse 1.5s ease-in-out infinite`
- Pulse ring: `@keyframes ring-pulse { 0% { transform: scale(1); opacity: 0.4; } 100% { transform: scale(2.5); opacity: 0; } }` on `.pulse-ring` with `animation: ring-pulse 1.5s ease-out infinite`
- Both dots: 12px circles, centered, `background: var(--cds-interactive);`

When progress is 0, show an indeterminate animation on the bar:
```css
.progress-fill[style*="width: 0%"] {
  width: 30% !important;
  animation: indeterminate 1.5s ease-in-out infinite;
}
@keyframes indeterminate {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(400%); }
}
```
