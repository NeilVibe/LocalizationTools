1. `create_idmap_index(cls, dim: int) -> "faiss.Index"` -- Creates `faiss.IndexIDMap2(faiss.IndexHNSWFlat(dim, cls.HNSW_M, faiss.METRIC_INNER_PRODUCT))` with same efConstruction/efSearch as create_index. IndexIDMap2 wraps HNSW and enables add_with_ids() and remove_ids().

2. `add_with_ids(cls, index, vectors, ids, normalize=True) -> None` -- Calls `index.add_with_ids(vectors, ids)` after normalization. vectors: np.ndarray (N x dim), ids: np.ndarray of int64.

3. `remove_ids(cls, index, ids) -> int` -- Calls `index.remove_ids(np.array(ids, dtype=np.int64))`. Returns number of removed vectors. Log warning if removed count != expected.

4. `convert_to_idmap(cls, existing_index, dim: int) -> "faiss.Index"` -- Takes a plain IndexHNSWFlat, creates a new IndexIDMap2 wrapper, reconstructs all vectors from the old index and adds them with sequential IDs (0, 1, 2, ...). This is for migrating existing indexes on first use. Note: IndexHNSWFlat supports reconstruct() so use `index.reconstruct(i)` in a loop to get vectors.

5. Add a `ThreadSafeIndex` class inside faiss_manager.py:
```python
import threading

class ThreadSafeIndex:
    """Thread-safe wrapper around a FAISS index for concurrent access."""
    def __init__(self, index):
        self._index = index
        self._lock = threading.RLock()

    @property
    def index(self):
        return self._index

    def add_with_ids(self, vectors, ids):
        with self._lock:
            self._index.add_with_ids(vectors, ids)

    def remove_ids(self, ids):
        with self._lock:
            return self._index.remove_ids(ids)

    def search(self, query, k):
        with self._lock:
            return self._index.search(query, k)

    @property
    def ntotal(self):
        return self._index.ntotal
```

Add `import threading` at the top of the file.

Create `server/tools/ldm/indexing/inline_updater.py` with class `InlineTMUpdater`.

**Constructor:** `__init__(self, tm_id: int, data_dir: str = None)`
- Sets up paths (same structure as TMSyncManager: `server/data/ldm_tm/{tm_id}/`)
- Default data_dir = `Path(__file__).parent.parent.parent.parent / "data" / "ldm_tm"` (same as sync_manager.py)
- Initializes `self._engine = None`, `self._ts_index = None` (ThreadSafeIndex), `self._whole_lookup = None`, `self._line_lookup = None`, `self._whole_mapping = None`, `self._whole_embeddings = None`

**Private methods:**

`_ensure_loaded(self)` -- Lazy-load everything:
- Embedding engine via `get_embedding_engine(get_current_engine_name())`, call `.load()` if not loaded
- FAISS index from `{tm_path}/faiss/whole.index` -- if exists, load via FAISSManager.load_index(). Check if it's an IDMap2 index (hasattr(index, 'id_map') or check type). If it's plain IndexHNSWFlat, call FAISSManager.convert_to_idmap() to migrate and save the migrated index back. Wrap in ThreadSafeIndex.
- If no index file exists, create new via FAISSManager.create_idmap_index(self._engine.dimension), wrap in ThreadSafeIndex.
- Load whole_lookup.pkl, line_lookup.pkl (default to empty dicts if not found)
- Load whole_mapping.pkl (default to empty list if not found)
- Load whole.npy (default to None if not found)

**Public methods:**

`add_entry(self, entry_id: int, source_text: str, target_text: str, string_id: str = None) -> None`
1. `self._ensure_loaded()`
2. Encode: `text = normalize_for_embedding(source_text)`, `embedding = self._engine.encode([text], normalize=True)`, convert to float32 ndarray
3. Normalize vector: `FAISSManager.normalize_vectors(embedding)`
4. FAISS add: `self._ts_index.add_with_ids(embedding, np.array([entry_id], dtype=np.int64))`
5. Update whole_lookup (copy pattern from sync_manager._incremental_sync lines 292-315):
   - `normalized = normalize_for_hash(source_text)`
   - `entry_data = {"entry_id": entry_id, "source_text": source_text, "target_text": target_text, "string_id": string_id}`
   - If not in lookup and has string_id: `{"variations": [entry_data], "source_text": source_text}`
   - If not in lookup and no string_id: just entry_data
   - If already in lookup: append to variations list
6. Update line_lookup (copy pattern from sync_manager._incremental_sync lines 317-336)
7. Append to whole_mapping list
8. Append embedding to whole_embeddings (np.vstack or create new)
9. `self._persist()`
10. Log: `logger.info(f"Inline add: tm_id={self.tm_id}, entry_id={entry_id}, source={source_text[:30]}...")`

`update_entry(self, entry_id: int, source_text: str, target_text: str, string_id: str = None, old_source_text: str = None) -> None`
1. If old_source_text provided, remove old data first
2. Remove old vector: `self._ts_index.remove_ids(np.array([entry_id], dtype=np.int64))`
3. Remove old hash entries: normalize_for_hash(old_source_text), delete from whole_lookup
4. Remove old line entries from line_lookup
5. Remove from whole_mapping (filter out entry_id)
6. Then add new data (same as add_entry steps 2-8)
7. `self._persist()`

`remove_entry(self, entry_id: int, source_text: str) -> None`
1. `self._ensure_loaded()`
2. Remove vector: `self._ts_index.remove_ids(np.array([entry_id], dtype=np.int64))`
3. Remove from whole_lookup using normalize_for_hash(source_text)
4. Remove from line_lookup for all lines of source_text
5. Remove from whole_mapping (filter out entry_id)
6. Rebuild whole_embeddings without this entry (filter by mapping index)
7. `self._persist()`

`add_entries_batch(self, entries: List[dict]) -> None`
- entries: list of {"id": int, "source_text": str, "target_text": str, "string_id": str|None}
1. `self._ensure_loaded()`
2. Batch encode: `texts = [normalize_for_embedding(e["source_text"]) for e in entries]`, `embeddings = self._engine.encode(texts, normalize=True)`
3. Normalize: `FAISSManager.normalize_vectors(embeddings)`
4. Batch FAISS: `ids = np.array([e["id"] for e in entries], dtype=np.int64)`, `self._ts_index.add_with_ids(embeddings, ids)`
5. Bulk update whole_lookup and line_lookup (loop over entries, same patterns)
6. Extend whole_mapping
7. Append/create whole_embeddings
8. Single `self._persist()` at end

`_persist(self) -> None`
- Ensure directories exist: `{tm_path}/faiss/`, `{tm_path}/hash/`, `{tm_path}/embeddings/`
- `FAISSManager.save_index(self._ts_index.index, faiss_path)`
- `pickle.dump(self._whole_lookup, ...)` to whole_lookup.pkl
- `pickle.dump(self._line_lookup, ...)` to line_lookup.pkl
- `pickle.dump(self._whole_mapping, ...)` to whole_mapping.pkl
- `np.save(...)` whole_embeddings to whole.npy
- Use `pickle.HIGHEST_PROTOCOL` (same as sync_manager)
