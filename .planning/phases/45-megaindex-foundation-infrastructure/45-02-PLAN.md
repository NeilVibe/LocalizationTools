**ai_capability_service.py:**
- `from __future__ import annotations` at top.
- `from loguru import logger` (never print).
- Create `AICapabilityService` class with:
  - `__init__(self)`: `self._capabilities: Dict[str, str] = {}` (values: "available", "unavailable", "checking"), `self._last_check: float = 0.0`
  - `check_all(self) -> Dict[str, str]`: Probes each engine and returns dict. Each probe is wrapped in try/except (never crashes):
    - `"embeddings"`: try `from model2vec import StaticModel; StaticModel.from_pretrained` check (just import, don't load). Also check `Model2VecEngine._find_local_model_path()` for bundled model. If model2vec importable -> "available", else -> "unavailable".
    - `"semantic_search"`: try `import faiss; faiss.IndexFlatIP(1)` -> "available", except -> "unavailable".
    - `"ai_summary"`: try `import httpx; resp = httpx.get("http://localhost:11434/api/tags", timeout=2.0); resp.status_code == 200` -> "available" (Ollama running), else -> "unavailable".
    - `"tts"`: check if `shutil.which("qwen-tts")` is not None, OR check for the TTS endpoint at localhost. If found -> "available", else -> "unavailable".
    - `"image_gen"`: check if `os.environ.get("FAL_KEY")` is set -> "available", else -> "unavailable".
  - `get_capabilities(self) -> Dict[str, str]`: returns cached `self._capabilities`. If empty, calls `check_all()` first.
  - `is_available(self, engine: str) -> bool`: returns `self._capabilities.get(engine) == "available"`.
  - `get_status(self) -> dict`: returns `{"capabilities": self._capabilities, "last_check": self._last_check, "light_mode": is_light_mode()}`.
- Import `is_light_mode` from `server.tools.shared.embedding_engine`.
- Singleton: `_service_instance` + `get_ai_capability_service()`.

**capabilities.py:**
- Create FastAPI router with prefix `""` (will be mounted at `/api/ldm/capabilities`).
- `GET /` -> calls `get_ai_capability_service().get_capabilities()`, returns JSON: `{"capabilities": {...}, "light_mode": bool, "last_check": float}`.
- `POST /refresh` -> calls `get_ai_capability_service().check_all()`, returns same format. Forces re-probe.

**routes/__init__.py:**
- Add import and `app.include_router(capabilities_router, prefix="/api/ldm/capabilities", tags=["capabilities"])`.
