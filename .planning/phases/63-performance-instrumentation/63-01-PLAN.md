  <action>
Create server/utils/perf_timer.py with:

1. **PerfTimer context manager** — measures wall-clock duration in ms using time.perf_counter(). Usage:
   ```python
   with PerfTimer("faiss_search") as t:
       results = index.search(query, k)
   # t.duration_ms available after exit
   ```
   On exit, it calls record_metric(operation, duration_ms, **extra_fields) automatically, AND emits a structured loguru log line: `logger.info("perf | op={operation} | duration_ms={duration_ms:.1f} | {extra_fields}")`.

2. **record_metric(operation: str, duration_ms: float, **kwargs)** — stores metric in a thread-safe in-memory ring buffer (collections.deque with maxlen=1000 per operation). Each record: `{"operation": str, "duration_ms": float, "timestamp": float, **kwargs}`. Use a module-level dict[str, deque] with threading.Lock for thread safety.

3. **get_metrics_summary() -> dict** — computes p50/p95/max/count/avg for each operation from the ring buffer. Returns `{"operations": {"faiss_search": {"p50": ..., "p95": ..., "max": ..., "count": ..., "avg": ...}, ...}}`. Use numpy for percentile calculations (already a project dependency). If deque is empty for an operation, return zeros.

4. **perf_timer(operation: str, **extra) -> PerfTimer** — convenience function that returns a PerfTimer pre-configured with operation name and extra fields to log (e.g., `perf_timer("upload", file_size_bytes=1024)`).

Use loguru logger (NOT print, NOT stdlib logging). Keep the module self-contained with no FastAPI dependencies.
  </action>

```python
with PerfTimer("embedding_encode", batch_size=len(texts)):

```

**PERF-02 — faiss_manager.py:** In `FAISSManager.search()` (classmethod ~line 166) and `ThreadSafeIndex.search()` (~line 430):
```python
with PerfTimer("faiss_search", k=k, index_size=index.ntotal):
    # existing search logic
```
Also in `FAISSManager.add_vectors()`:
```python
with PerfTimer("faiss_add_vectors", count=len(vectors)):
    # existing add logic
```

**PERF-03 — inline_updater.py:** In `add_entry()`, `update_entry()`, `remove_entry()`:
```python
with PerfTimer("tm_add_entry", tm_id=self.tm_id, entry_id=entry_id):
    # existing logic (embedding + index update)
```
(Use "tm_update_entry" and "tm_remove_entry" for the other two.)

**PERF-03 — tm_entries.py:** In the route-level `add_entry`, `update_entry`, `delete_entry` endpoints, wrap the entire handler body:
```python
with PerfTimer("tm_entry_add_endpoint"):
    # existing handler logic
```

**PERF-04 — merge.py:** In `preview_merge()`, wrap the execute_transfer call:
```python
with PerfTimer("merge_preview", match_mode=body.match_mode):
```

In `execute_merge()`, wrap the SSE generator's transfer call:
```python
with PerfTimer("merge_execute", match_mode=body.match_mode):
    result = await asyncio.to_thread(execute_transfer, ...)
```
