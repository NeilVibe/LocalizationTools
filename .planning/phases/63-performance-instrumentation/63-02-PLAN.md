**server/api/performance.py** — Create new file:

1. Create FastAPI APIRouter with tags=["Performance"] and prefix="/api/performance".

2. **GET /summary** endpoint:
   - Import `get_metrics_summary` from `server.utils.perf_timer`
   - Call `get_metrics_summary()` and return the result directly as JSON
   - Response model should include `operations` dict with per-operation stats
   - No authentication required (dev/diagnostic endpoint)
   - Add response_model using a Pydantic model:
     ```python
     class OperationStats(BaseModel):
         p50: float
         p95: float
         max: float
         count: int
         avg: float

     class PerformanceSummaryResponse(BaseModel):
         operations: dict[str, OperationStats]
     ```

3. **POST /reset** endpoint (optional but useful):
   - Import the metrics store and clear all deques
   - Returns `{"status": "reset", "operations_cleared": N}`
