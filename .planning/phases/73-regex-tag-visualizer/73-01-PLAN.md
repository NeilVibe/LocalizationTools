1. `detectTags(text)` - Returns array of segments: `[{text: string} | {tag: {label, type, color, raw, inner?}}]`
   - Uses single-pass approach: collect all matches from all 5 patterns, sort by position, build segments
   - Pattern priority (process in this order to handle overlaps):
     - Priority 1 `staticinfo`: `/\{Static[Ii]nfo:(\w+):([^#}]+)#([^}]+)\}/g` -> label=capture[2], inner=capture[3], type='staticinfo', color='green'
     - Priority 2 `param`: `/%(\d)#/g` -> label=`Param${capture[1]}`, type='param', color='purple'
     - Priority 3 `braced`: `/\{(?!Static[Ii]nfo:\w+:)([^}]+)\}/g` -> label=capture[1], type='braced', color='blue'
     - Priority 4 `escape`: `/\\(\w)/g` -> label=`\\${capture[1]}`, type='escape', color='grey'
       NOTE: Use `\w` (word chars: letters, digits, underscore) to match tmx_tools.py source exactly. This covers \n, \t, \1, \_, etc.
     - Priority 5 `desc`: `/&(?:amp;)?desc;/g` -> label='desc', type='desc', color='orange'
   - Overlap prevention: for each match, check if its start position falls within any already-collected match's [start, end) range. Skip if overlapping.
   - Return `[{text: text}]` for null/empty/non-string input

2. `hasTags(text)` - Fast boolean check using simple patterns (no capture groups):
   - `/%\d#/.test(text) || /\{[^}]+\}/.test(text) || /\\[\w]/.test(text) || /&(?:amp;)?desc;/.test(text)`
   NOTE: Use `\w` in the escape fast-path to match the detectTags escape pattern exactly.

```svelte
<script>
  import { detectTags, hasTags } from '$lib/utils/tagDetector.js';
  import ColorText from './ColorText.svelte';

  let { text = '' } = $props();

  // Fast path: skip detection for plain text
  let segments = $derived(hasTags(text) ? detectTags(text) : null);

  // Apply formatGridText ONLY to plain text segments
  // CRITICAL: Do NOT convert \n to newline here -- \n is a tag pill
  // Only convert &lt;br/&gt; to actual newlines
  function formatPlainText(t) {
    if (!t) return '';
    return t.replace(/&lt;br\/&gt;/g, '\n');
  }
</script>
```

Template structure:
- If `segments` is non-null (text has tags):
  - Wrap in `<span class="tag-text">`
  - `{#each segments as seg, i (i)}` (ALWAYS use keys per Svelte 5 rule)
  - If `seg.tag`: render `<span class="tag-pill tag-{seg.tag.color}" title={seg.tag.raw}>{seg.tag.label}</span>`
  - If `seg.tag.inner` exists (StaticInfo): render inner text via `<ColorText text={formatPlainText(seg.tag.inner)} />`
  - If `seg.text` (plain segment): render `<ColorText text={formatPlainText(seg.text)} />`
- Else (no tags, fast path): render `<ColorText text={formatPlainText(text)} />`
