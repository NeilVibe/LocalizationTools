    1. `from fastapi import APIRouter`
    2. `from pydantic import BaseModel`
    3. `from pathlib import Path`

    5. `from loguru import logger`

       ```python

       ```

    8. Define response model:
       ```python
       class PathValidationResponse(BaseModel):
           valid: bool
           error: str | None = None
           hint: str | None = None
           files_found: int = 0
           file_types: list[str] = []
       ```

    9. Define helper `def translate_wsl_path(path_str: str) -> str`:

         - Extract drive letter, lowercase it
         - Replace backslashes with forward slashes
         - Return `/mnt/{drive}/{rest_of_path}`
       - Otherwise return path_str unchanged

    10. Define `@router.post("/validate-path", response_model=PathValidationResponse)`:
        - Translate path with `translate_wsl_path(body.path)`
        - Check `Path(translated).exists()` — if not, return valid=False, error="Path does not exist"
        - Check `Path(translated).is_dir()` — if not, return valid=False, error="Path is not a directory"
        - Glob for `languagedata_*.*` in the directory
        - If no files found, return valid=False, error="No languagedata files found", hint="Expected files like languagedata_fre.xml or languagedata_fr.txt"
        - Otherwise return valid=True, files_found=len(files), file_types=[f.name for f in files]

    11. Register router in `server/main.py`: add `from server.api.settings import router as settings_router` and `app.include_router(settings_router)`
