  <action>
Create `locaNext/src/lib/components/common/AppModal.svelte` — a thin Svelte 5 wrapper around Carbon's Modal component.

This wrapper:
1. Imports Carbon Modal: `import { Modal } from 'carbon-components-svelte';`
2. Accepts ALL Carbon Modal props via `$props()` with defaults matching Carbon's defaults
3. Accepts TWO additional callback props: `onprimary` and `onsecondary` (both default to `() => {}`)
4. Accepts `onclose` callback prop (default `undefined`)
5. Renders `<Modal>` with all props spread, plus:
   - `on:click:button--primary={() => onprimary()}`
   - `on:click:button--secondary={(e) => onsecondary(e)}`
   - `on:close={(e) => onclose?.(e)}`
6. Passes through the default slot using `{@render children?.()}`
7. Uses `bind:open` to support `$bindable(false)` on the open prop

The component signature:
```svelte
<script>
  import { Modal } from 'carbon-components-svelte';

  let {
    open = $bindable(false),
    size = undefined,
    danger = false,
    modalHeading = undefined,
    primaryButtonText = '',
    primaryButtonDisabled = false,
    secondaryButtonText = '',
    hasForm = false,
    hasScrollingContent = false,
    selectorPrimaryFocus = '[data-modal-primary-focus]',
    preventCloseOnClickOutside = false,
    shouldSubmitOnEnter = true,
    onprimary = () => {},
    onsecondary = () => {},
    onclose = undefined,
    children,
    ...restProps
  } = $props();
</script>

<Modal
  bind:open
  {size}
  {danger}
  {modalHeading}
  {primaryButtonText}
  {primaryButtonDisabled}
  {secondaryButtonText}
  {hasForm}
  {hasScrollingContent}
  {selectorPrimaryFocus}
  {preventCloseOnClickOutside}
  {shouldSubmitOnEnter}
  {...restProps}
  on:click:button--primary={onprimary}
  on:click:button--secondary={onsecondary}
  on:close={(e) => onclose?.(e)}
>
  {@render children?.()}
</Modal>
```

This is the ONLY file in the codebase that uses `on:` syntax for Carbon Modal events. All other files will use the Svelte 5 callback props `onprimary` and `onsecondary`.
