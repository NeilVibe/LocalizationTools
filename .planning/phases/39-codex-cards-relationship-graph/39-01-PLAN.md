**Parallax hover logic:**
- Use `$effect` to attach mousemove/mouseleave listeners to the card element (use `bind:this`)
- On mousemove: calculate mouse position relative to card center, compute rotateX (max +/-5deg) and rotateY (max +/-5deg)
- Apply `transform: perspective(800px) rotateX({ry}deg) rotateY({rx}deg)` to card
- Apply `transform: translate({-rx * 2}px, {-ry * 2}px)` to `.card-hero-image` (opposite parallax)
- On mouseleave: reset both transforms to identity with 200ms transition
- Respect `prefers-reduced-motion` -- skip parallax if user prefers reduced motion

**Card layout (vertical card, not horizontal like current):**
- Hero image area: 160px height, full width, `object-fit: cover`
- Image shimmer: Use CSS transition on opacity (0 -> 1) when image loads via `onload` event
- Shimmer placeholder: `background: linear-gradient(90deg, rgba(255,255,255,0.03) 25%, rgba(255,255,255,0.08) 50%, rgba(255,255,255,0.03) 75%); background-size: 200% 100%; animation: shimmer 1.5s ease-in-out infinite`
- Type badge: absolute positioned top-right (8px, 8px), use Carbon Tag with TYPE_COLORS
- Card body: padding 12px, flex column
  - Name: 0.875rem, font-weight 600, color var(--cds-text-01), single line ellipsis
  - Description: 0.75rem, color var(--cds-text-02), 2 lines max with `-webkit-line-clamp: 2`
  - Stats row: flex row at bottom, show related_entities count as "X refs" badge

**Entrance stagger:**
- CSS custom property `--card-index` set via `style="--card-index: {index}"` on the card element
- Use CSS transition (NOT animation) triggered by a `.visible` class added after mount via `$effect` with requestAnimationFrame
- Before visible: `opacity: 0; transform: translateY(16px)`
- After visible: `opacity: 1; transform: translateY(0); transition-delay: calc(var(--card-index) * 60ms)`

**Image resolution order (same as current CodexPage):**
1. `entity.ai_image_url` if set and not in failedImages
2. `entity.image_texture` thumbnail if set and not in failedImages
3. PlaceholderImage component fallback

Use `loading="lazy"` on all img elements. Call `onfailimage(imageKey)` on error.

**prefers-reduced-motion media query:**
```css
@media (prefers-reduced-motion: reduce) {
  .codex-card { transition: none; }
  .card-hero-image { transition: none; }
}
```
