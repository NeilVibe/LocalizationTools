2. **POST /execute endpoint** (add after the preview endpoint):
   ```
   @router.post("/execute")
   async def execute_merge(body: MergeExecuteRequest):
   ```

   Implementation steps:

   a. **Check merge guard:** If `_merge_in_progress` is True, return `JSONResponse(status_code=409, content={"error": "A merge is already in progress"})`. Import JSONResponse from `fastapi.responses`.

   b. **Validate match_mode:** Same validation as preview -- if `body.match_mode not in MATCH_MODES`, return 422.

   c. **Translate paths:** Apply `translate_wsl_path()` to all 3 paths.

   d. **Create asyncio.Queue:** `queue: asyncio.Queue = asyncio.Queue()`

   e. **Define callbacks:**
   - `progress_cb(message: str)`: calls `queue.put_nowait({"event": "progress", "data": message})`
   - `log_cb(message: str, level: str = "info")`: calls `queue.put_nowait({"event": "log", "data": json.dumps({"message": message, "level": level})})`

   CRITICAL: Use `put_nowait()` NOT `put()` -- the callbacks run in a sync thread (Pitfall 1 from research).

   f. **Define async run_transfer():**
   - Set `global _merge_in_progress; _merge_in_progress = True`
   - try block:
     - If `body.multi_language`: call `await asyncio.to_thread(execute_multi_language_transfer, source_path=..., target_path=..., export_path=..., match_mode=..., only_untranslated=..., stringid_all_categories=..., dry_run=False, progress_callback=progress_cb, log_callback=log_cb)`
     - Else: call `await asyncio.to_thread(execute_transfer, ...)` with same pattern
     - On success: `queue.put_nowait({"event": "complete", "data": json.dumps(result, default=str)})`
       CRITICAL: Use `default=str` in json.dumps -- QT returns Path objects (Pitfall 2).
   - except block: `queue.put_nowait({"event": "error", "data": str(exc)})`
   - finally block: `_merge_in_progress = False`

   g. **Define async event_generator():**
   - `task = asyncio.create_task(run_transfer())`
   - Loop:
     - `try: msg = await asyncio.wait_for(queue.get(), timeout=30.0)`
     - `except asyncio.TimeoutError:` yield `{"event": "ping", "data": "keepalive"}` and continue
     - `yield msg`
     - `if msg["event"] in ("complete", "error"): break`
   - `await task` (ensure task finishes cleanly)

   h. **Return:** `return EventSourceResponse(event_generator())`

3. **Add `from fastapi.responses import JSONResponse`** to imports if not already present.

CRITICAL anti-patterns:
- NEVER use `queue.put()` from sync callback (it's a coroutine, will fail in thread)
- NEVER forget `default=str` in `json.dumps(result, ...)` -- Path objects break serialization
- NEVER forget to reset `_merge_in_progress = False` in finally block
- NEVER forget WSL path translation on all 3 paths

   ```python
   def parse_sse_events(response_text: str) -> list[dict]:
       events = []
       current = {}
       for line in response_text.split("\n"):
           if line.startswith("event:"):
               current["event"] = line[6:].strip()
           elif line.startswith("data:"):
               current["data"] = line[5:].strip()
           elif line == "" and current:
               events.append(current)
               current = {}
       if current:
           events.append(current)
       return events
   ```
