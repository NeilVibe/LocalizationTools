Create `server/tools/ldm/services/gamedata_context_service.py` with class `GameDataContextService`:

1. **Reverse index builder** -- `build_reverse_index(folder_tree_data: FolderTreeDataResponse)`:
   - Walk all TreeNodes from folder_tree_data.files[].roots recursively
   - For each node, scan ALL attribute values for cross-reference patterns:
     - Attributes ending in Key, Id, NodeId (explicit cross-refs)
     - Check each value against the whole_lookup hashtable from GameDataIndexer
   - Build `_reverse_index: Dict[str, List[Dict]]` mapping `target_node_id -> [{source_node_id, source_tag, source_attr, source_file}]`
   - Store as instance attribute, called once after folder load

2. **Cross-ref resolver** -- `get_cross_refs(node_id: str, node: TreeNode) -> Dict`:
   - Forward refs: scan node.attributes for values in whole_lookup, return [{target_node_id, target_tag, target_name, via_attr}]
   - Backward refs: lookup node_id in _reverse_index, return [{source_node_id, source_tag, source_name, via_attr}]
   - Group by entity type (tag)

3. **Related entities** -- `get_related(node: TreeNode, top_k=5) -> List[Dict]`:
   - Get entity name from first editable attr or Key attr
   - Call GameDataSearcher.search(entity_name, top_k=top_k, threshold=0.85)
   - Filter out self (exclude results with same node_id)
   - Return results with tier_name badge info

4. **TM suggestions from language data** -- `async def get_tm_suggestions(node: TreeNode, row_repo, threshold=0.50, max_results=5) -> List[Dict]`:
   - Check if entity has a StrKey attribute (look for "StrKey", "Key" ending attrs in node.attributes)
   - If no StrKey found, return empty list (TM suggestions only relevant for entities with language data linkage)
   - Get the entity name/description from the first editable attr value
   - Call `row_repo.suggest_similar(source=entity_name_or_desc, threshold=threshold, max_results=max_results)` -- same repository method used by /tm/suggest endpoint
   - Return list of `{source, target, similarity, file_name}` -- the existing TM suggestion format
   - On error (e.g., offline mode with SQLite where pg_trgm unavailable), return empty list gracefully

5. **Media resolver** -- `get_media(node: TreeNode) -> Dict`:
   - Check for texture attributes: TextureName, IconTexture, Texture, ImagePath in node.attributes
   - If found, return {has_image: true, texture_name: value, thumbnail_url: `/api/ldm/mapdata/thumbnail/{value}`}
   - Check for voice attributes: VoiceId, SoundId, AudioFile in node.attributes
   - If found, return {has_audio: true, voice_id: value, stream_url: `/api/ldm/mapdata/audio/stream/{value}`}
   - If neither, return {has_image: false, has_audio: false}

6. **Singleton** -- `get_gamedata_context_service()` function returning module-level instance
