Build Trigger File - LocaNext QA Build System
==============================================

USAGE:
------
Add a line with the build command, then commit & push to main.
GitHub Actions will automatically build when it detects the new line.

FORMATS:
  Build / Build QA    → ALL tests (~1000+), installer (~150MB) [DEFAULT]
  TROUBLESHOOT        → Resume failed build, debug mode

Architecture: Small installer + first-run setup downloads deps + AI model

WORKFLOW:
1. Add build command to this file
2. Commit and push to main
3. GitHub Actions will automatically build

QA MODE: All 1000+ tests run. Workers technology makes it fast.
